主要讲了两部分吧。一部分是作用域分为global, local, system。另一部分讲了设置user.name和user.email  

### 1.作用域
* global的作用域是所有仓库
* local的作用域是单个仓库
* system:对系统所有登录的用户有效(这个基本不用)
